## A6 – Discover cryptographic implementations used offline

### Objective

Identify five different cryptographic implementations used offline. The goal is to understand how encryption is used to protect data and systems without requiring an internet connection.

---

### Method

1. Defined scope:

* Focused on cryptographic uses that work without internet.
* Included storage, authentication, and device-level security.

2. Research approach:

* Looked at common devices like laptops, USB drives, and smart cards.
* Selected examples with different uses of cryptography.

3. Selection criteria:

* Must work offline.
* Each example must use cryptography in a different way.

---

### Findings

#### 1. Full Disk Encryption (e.g., laptop encryption)

* Used in laptops and desktops.

**How it works:**

* Encrypts all data on the hard drive.
* Requires a password to unlock the system.

**Weakness:**

* If password is weak, data can be accessed.

---

#### 2. Encrypted USB Drives

* Used to store sensitive files.

**How it works:**

* Data on the USB is encrypted.
* Requires a password or key to access files.

**Weakness:**

* Can be lost or stolen.

---

#### 3. Smart Cards (Chip-based Cards)

* Used in ID cards and access systems.

**How it works:**

* Stores encrypted data inside a chip.
* Requires a reader to verify information.

**Weakness:**

* Needs special hardware to use.

---

#### 4. ATM PIN Encryption

* Used in ATM machines.

**How it works:**

* PIN entered by user is encrypted before processing.
* Protects sensitive input from being exposed.

**Weakness:**

* Physical attacks (like skimming) can still occur.

---

#### 5. Offline Password Vault (Encrypted File)

* Used in local password manager apps.

**How it works:**

* Stores passwords in an encrypted file on the device.
* Accessed using a master password.

**Weakness:**

* If device is compromised, vault may be at risk.

---

### Mitigations (high-level recommendations)

1. Use strong passwords:

* Protect encrypted systems with complex passwords.

2. Secure physical devices:

* Keep USBs and smart cards safe.

3. Regular backups:

* Prevent data loss from damage or theft.

4. Update systems:

* Keep encryption software up to date.

---

### Reflection

This task showed that cryptography is not only used online but also plays an important role offline. It helps protect stored data and secure devices even without internet access. These methods are widely used in everyday technology and are important for maintaining privacy and security.
