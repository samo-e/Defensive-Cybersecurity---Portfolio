## A12 – Discover 5 unique offline security tools

### Objective

Identify five different offline security tools used in daily life. The goal is to understand how physical and non-digital tools help protect people, property, and information without using the internet.

---

### Method

1. Defined scope:

* Focused on physical and offline security tools used in homes, offices, and public places.
* No internet-based or online systems included.

2. Research approach:

* Observed common security tools used in daily life (homes, shops, vehicles).
* Selected tools with different methods of protection.

3. Selection criteria:

* Tools must work without internet.
* Each tool must serve a different security purpose.

---

### Findings

#### 1. Door Lock (Key-based Lock)

* Used in homes, offices, and rooms.

**How it works:**

* A physical key is used to lock and unlock the door.

**Weakness:**

* Keys can be lost, stolen, or copied.

---

#### 2. Padlock

* Used for gates, lockers, and storage units.

**How it works:**

* A lock that can be attached and removed using a key or code.

**Weakness:**

* Can be broken with force or tools if not strong enough.

---

#### 3. CCTV Camera (Offline Recording)

* Used in homes and businesses for monitoring.

**How it works:**

* Records video and stores it locally (e.g., on a memory card or DVR).

**Weakness:**

* Does not prevent crime directly, only records it.
* Can be damaged or avoided.

---

#### 4. Safe (Locker/Security Box)

* Used to store valuables like money, documents, and jewelry.

**How it works:**

* Locked container that requires a key or code to open.

**Weakness:**

* Can be stolen if not fixed in place.
* Weak safes can be broken.

---

#### 5. Security Alarm (Local Alarm System)

* Used in homes and shops.

**How it works:**

* Triggers a loud sound when someone enters without permission.

**Weakness:**

* Can be ignored or disabled.
* May cause false alarms.

---

### Mitigations (high-level recommendations)

1. Use strong and high-quality tools:

* Choose durable locks and safes.

2. Combine multiple tools:

* Example: lock + alarm + camera for better security.

3. Keep keys and codes secure:

* Do not share access with unknown people.

4. Regularly check devices:

* Make sure alarms and cameras are working properly.

---

### Reflection

This task showed that security is not only digital but also physical. Everyday tools like locks, safes, and alarms play a big role in protecting property and safety. While these tools are simple, they are still very important. Using multiple layers of security together gives better protection than relying on just one method.
