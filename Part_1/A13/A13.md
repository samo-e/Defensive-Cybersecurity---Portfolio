## A13 – Discover 5 unique online security tools

### Objective

Identify five different online security tools that help protect users, systems, and data on the internet. The goal is to understand how security is maintained in online environments using different types of tools.

---

### Method

1. Defined scope:

* Focused only on tools that require internet access.
* Included tools used by individuals and organizations.

2. Research approach:

* Looked at commonly used online security services.
* Selected tools with different purposes (e.g., protection, privacy, detection).

3. Selection criteria:

* Must work online (cloud-based or internet-dependent).
* Each tool must provide a unique function.

---

### Findings

#### 1. Cloud-based Antivirus (e.g., web security platforms)

* Protects devices using cloud threat databases.

**How it works:**

* Scans files and activity using online servers.
* Detects threats using real-time updates.

**Weakness:**

* Requires internet connection to work effectively.

---

#### 2. Online Password Manager

* Stores and manages passwords securely in the cloud.

**How it works:**

* Saves encrypted passwords online.
* Allows access from multiple devices.

**Weakness:**

* If the master password is compromised, all accounts are at risk.

---

#### 3. Virtual Private Network (VPN)

* Protects privacy and hides user location online.

**How it works:**

* Encrypts internet traffic.
* Sends it through a secure server.

**Weakness:**

* Trust depends on the VPN provider.
* May slow down internet speed.

---

#### 4. Web Application Firewall (WAF)

* Protects websites from online attacks.

**How it works:**

* Filters incoming traffic.
* Blocks malicious requests (e.g., SQL injection).

**Weakness:**

* May block legitimate users if not configured properly.

---

#### 5. Email Spam Filter (Cloud-based)

* Protects users from phishing and spam emails.

**How it works:**

* Scans incoming emails using online databases.
* Blocks or marks suspicious messages.

**Weakness:**

* Some phishing emails may still bypass filters.

---

### Mitigations (high-level recommendations)

1. Use trusted providers:

* Choose well-known and reliable security tools.

2. Enable multi-factor authentication:

* Adds extra protection to accounts.

3. Keep software updated:

* Ensure tools are running the latest versions.

4. Stay aware of threats:

* Do not rely only on tools—user awareness is important.

---

### Reflection

This task showed that online security tools are essential for protecting users on the internet. These tools work together to prevent attacks, protect data, and maintain privacy. However, they are not perfect, and users must still follow safe practices. A combination of tools and user awareness provides the best protection.
