## A11 – Discover 5 unique access control devices

### Objective

The objective of this activity was to identify five different access control devices used in real-world systems to restrict entry or access. I focused on common examples from hotels, offices, homes, phones, and public systems because these show how access control works in everyday environments around Perth and elsewhere.

---

### Findings

* **Key card readers**

  Key card readers are commonly used in hotels, offices, and universities to control entry into buildings or rooms. Users are given a card that contains a small embedded chip, and when the card is tapped or swiped on the reader, the system checks if the card is authorised. If valid, the door unlocks, making it a convenient and fast access method.

  I saw these at UWA buildings and hotels near Perth city. They are better than metal keys because cards can be deactivated if lost.

* **Fingerprint scanners**

  Fingerprint scanners are a type of biometric security device that uses a person's unique fingerprint to verify identity. These are widely used in smartphones and secure facilities. The system scans the fingerprint and compares it with stored data, allowing access only if there is a match, which makes it more secure than traditional keys.

  My phone uses this, and I have seen them at some office doors. They are quick but can fail if your fingers are wet or dirty.

* **PIN code keypads**

  PIN code keypads are used in places like door locks, ATMs, and alarm systems where users must enter a numeric code to gain access. The system checks the entered code against stored data, and if correct, access is granted. This method is simple and does not require carrying a physical key.

  These are everywhere, from apartment gates to bank ATMs in Perth. The downside is shoulder surfing if someone watches you enter the code.

* **Facial recognition systems**

  Facial recognition systems use cameras and AI to identify a person based on their facial features. These systems are used in modern smartphones and high-security environments. The camera scans the face and compares it with stored data to allow or deny access, making it a contactless and fast method.

  Newer phones like iPhone use this, and some airports have it too. It works well in good lighting but can have false positives with similar-looking people.

* **Smart locks**

  Smart locks are modern locking systems that can be controlled using a mobile app, Bluetooth, or Wi-Fi. They are commonly used in smart homes and rental properties. Users can unlock doors remotely or grant temporary access to others, making them flexible and convenient.

  I have seen these in Airbnb rentals around Perth. They are handy for guests but need good battery and secure app authentication.

---

### Mitigations

Security can be improved by combining multiple access control methods, securing credentials, and regularly updating systems. For example, using biometrics plus PIN is stronger than either alone.

It is also important to monitor access logs and change codes regularly. Physical tampering is a risk, so robust installation matters too.

---

### Reflection

This task showed that access control systems are widely used and combining different methods improves security. From cards to biometrics, each has strengths and weaknesses.

I also realised that no system is perfect — lost cards, dirty fingerprints, or weak PINs can all be problems. Good habits and layered controls are key.
