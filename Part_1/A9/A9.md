## A9 – Discover privacy techniques used online

### Objective

Identify five different privacy techniques used online. The goal is to understand how users protect their personal information, identity, and activity while using the internet.

---

### Method

1. Defined scope:

* Focused on techniques used while browsing, communicating, or using online services.
* Included both tools and user practices.

2. Research approach:

* Looked at common privacy practices used by individuals online.
* Selected techniques that reduce data exposure or tracking.

3. Selection criteria:

* Must be used online.
* Each technique must protect privacy in a different way.

---

### Findings

#### 1. Using a Virtual Private Network (VPN)

* Helps hide user identity and location online.

**How it works:**

* Encrypts internet traffic.
* Routes it through a secure server.

**Weakness:**

* Requires trust in the VPN provider.

---

#### 2. Enabling Two-Factor Authentication (2FA)

* Adds extra protection to online accounts.

**How it works:**

* Requires a second step (e.g., code on phone) after password.

**Weakness:**

* Can be inconvenient.
* SMS-based 2FA can be less secure.

---

#### 3. Using Strong and Unique Passwords

* Prevents unauthorized access to accounts.

**How it works:**

* Each account uses a different, complex password.

**Weakness:**

* Hard to remember without a password manager.

---

#### 4. Adjusting Privacy Settings on Websites

* Controls what data is shared.

**How it works:**

* Users limit profile visibility and data sharing options.

**Weakness:**

* Settings can be confusing or change over time.

---

#### 5. Using Private/Incognito Browsing Mode

* Reduces local tracking on devices.

**How it works:**

* Browser does not save history, cookies, or form data.

**Weakness:**

* Does not hide activity from websites or internet providers.

---

### Mitigations (high-level recommendations)

1. Use multiple techniques together:

* Example: VPN + strong passwords + 2FA.

2. Stay updated:

* Keep apps and browsers updated for better privacy.

3. Be cautious with sharing:

* Avoid posting sensitive personal information online.

4. Review settings regularly:

* Check privacy settings on accounts often.

---

### Reflection

This task showed that online privacy depends on both tools and user behaviour. While tools like VPNs and 2FA provide strong protection, users must also make smart choices about what they share. Using multiple techniques together gives better privacy and reduces risk.
