## A11 – Discover 5 unique access control devices

### Objective

Identify five different access control devices used in real-world systems.

---

### Findings

* Key card readers are commonly used in hotels, offices, and universities to control entry into buildings or rooms. Users are given a card that contains a small embedded chip, and when the card is tapped or swiped on the reader, the system checks if the card is authorised. If valid, the door unlocks, making it a convenient and fast access method.

* Fingerprint scanners are a type of biometric security device that uses a person’s unique fingerprint to verify identity. These are widely used in smartphones and secure facilities. The system scans the fingerprint and compares it with stored data, allowing access only if there is a match, which makes it more secure than traditional keys.

* PIN code keypads are used in places like door locks, ATMs, and alarm systems where users must enter a numeric code to gain access. The system checks the entered code against stored data, and if correct, access is granted. This method is simple and does not require carrying a physical key.

* Facial recognition systems use cameras and AI to identify a person based on their facial features. These systems are used in modern smartphones and high-security environments. The camera scans the face and compares it with stored data to allow or deny access, making it a contactless and fast method.

* Smart locks are modern locking systems that can be controlled using a mobile app, Bluetooth, or Wi-Fi. They are commonly used in smart homes and rental properties. Users can unlock doors remotely or grant temporary access to others, making them flexible and convenient.

---

### Mitigations (high-level recommendations)

Security can be improved by combining multiple access control methods, securing credentials, and regularly updating systems.

---

### Reflection

This task showed that access control systems are widely used and combining different methods improves security.
