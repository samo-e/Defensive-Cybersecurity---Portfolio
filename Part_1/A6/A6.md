## A6 – Discover cryptographic implementations used offline

### Objective

The objective of this activity was to identify five different cryptographic implementations used offline and understand how they help protect data and systems without needing an internet connection. I focused on common devices such as laptops, USB drives, smart cards, ATMs, and password managers because these are the places where offline cryptography is used in everyday life.

---

### Findings

* **Full disk encryption on laptops**

  Full disk encryption is commonly used on laptops and computers to protect all stored data. It works by encrypting the entire hard drive, meaning the data cannot be accessed unless the correct password is entered during startup. This ensures that even if the device is stolen, the data remains protected.

  I noticed that this is very important for students and professionals who carry laptops around, because it protects documents, photos, and other personal files. Tools like BitLocker or FileVault make it easy to set up on Windows or Mac.

* **Encrypted USB drives**

  Encrypted USB drives are used to securely store sensitive files such as documents or backups. These devices require a password or authentication before allowing access, which helps prevent unauthorized users from reading the data if the USB is lost or stolen.

  In my view, this is a simple but effective way to protect data on the move. Many USB drives now come with built-in encryption, and it is especially useful for sharing files without risking exposure.

* **Smart cards with embedded chips**

  Smart cards, such as ID cards with embedded chips, use cryptography to securely store and verify user information. When inserted into a reader, the card communicates encrypted data to confirm identity, making them useful for secure access systems.

  This is important because it allows offline authentication without needing a network. For example, some university ID cards or employee badges use this technology to unlock doors or vending machines.

* **ATM PIN protection**

  ATM machines use cryptography to protect user PINs during transactions. When a user enters their PIN, it is encrypted before being processed by the system, reducing the risk of interception or exposure.

  I found this interesting because it shows how cryptography protects financial data even in a completely offline environment. The PIN is never sent in plain text, which makes shoulder-surfing attacks less effective.

* **Offline password vaults**

  Offline password vaults store user passwords in an encrypted file on a local device. Access to the vault is controlled by a master password, allowing users to securely manage multiple passwords without needing an internet connection.

  This is a great example of cryptography for personal use. Tools like KeePass allow you to generate and store strong passwords locally, which is much safer than remembering weak ones.

---

### Mitigations

To keep these cryptographic systems secure, it is important to use strong and unique passwords, as weak passwords can make encryption ineffective. Physical security should also be considered, since devices like USB drives and smart cards can be lost or stolen. Regular backups are important to prevent permanent data loss, especially when encryption is involved.

In addition, keeping encryption software up to date helps protect against vulnerabilities and ensures better overall security. Sometimes people forget that encryption is only as strong as the key that protects it, so good key management is essential.

---

### Reflection

This task showed that cryptography is not only used online but also plays an important role offline. It helps protect stored data and secure devices even without internet access. These methods are widely used in everyday technology and are important for maintaining privacy and security.

I also realised that offline cryptography is just as critical as online, because most data is stored locally at some point. Whether it is a laptop, USB drive, or password vault, these tools make sure data stays safe even if the device is offline or stolen.
