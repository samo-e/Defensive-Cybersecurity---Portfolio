## A13 – Discover 5 unique online security tools

### Objective

Identify five different online security tools. I researched at password managers, antivirus software, browser security tools, VPN services, and cloud security tools because these are common ways people protect their devices and accounts while using the internet.

---

### Findings

* **Password managers**

  Password managers are used to create, store, and auto-fill strong passwords for online accounts. Instead of remembering many passwords, users only need one master password, which makes account management much easier and safer. They also reduce the temptation to reuse weak passwords across multiple sites.

  I think this is one of the most useful online security tools because many people still use the same password everywhere. A password manager helps fix that problem in a very practical way.

* **Antivirus software**

  Antivirus software protects devices from malware, spyware, ransomware, and other harmful files that may come from downloads, email attachments, or websites. It scans files and warns users before risky software is installed or opened.

  In my view, antivirus is still important even though browsers and operating systems have improved a lot. It adds an extra layer of protection, especially for users who download files often.

* **Browser security tools**

  Browser security tools help block phishing sites, malicious ads, trackers, and unsafe extensions. Some are built into the browser itself, while others come as add-ons that scan links in real time and warn users before they visit suspicious pages.

  I noticed that this kind of tool is very useful because a lot of threats start in the browser. If a bad link is blocked early, the attack can be stopped before it becomes a bigger issue.

* **VPN services**

  VPN helps encrypts a user’s internet traffic and hides their IP address, helping protect privacy on public networks. They are commonly used on laptops and mobile phones specially if people connect at cafe, airports, and use shared Wi-Fis.

  VPN is not perfect, but it does give better privacy in many situations. It also reduces the chance of someone sniffing network traffic on the same network that could steal your data.

* **Cloud security tools**

  Cloud security tools are used to protect online accounts, cloud apps, and stored data. They may include identity protection, access monitoring, threat detection, and browser-based controls that limit risky behaviour in cloud environments.

  I found this interesting because many people now keep documents and photos in cloud services (example: Google Docs). If the cloud account is weak, the whole data store becomes vulnerable, so these tools are really useful.

---

### Mitigations

Online security can be improved by combining multiple tools instead of relying on only one. For example, a password manager, antivirus, and VPN together provide much better protection than any single tool alone.

Users should also keep all security tools updated and avoid installing random browser extensions. Security tools work best when they are used properly, and not just installed and forgotten.

---

### Reflection

This task showed me that online security is not just about one software or one setting. It is really a combination of tools that protect passwords, devices, browsers, and data.

I also realised that the best tools are the ones people actually use consistently. A simple tool used well is more effective than a powerful one that gets ignored.
