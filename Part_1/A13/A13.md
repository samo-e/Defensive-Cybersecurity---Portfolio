## A13 – Discover 5 unique online security tools

### Objective

Identify five online security tools used to protect users and systems.

---

### Findings

* Cloud-based antivirus tools protect devices by using online databases and real-time updates to detect malware. Instead of relying only on local files, these tools continuously connect to cloud servers to identify new threats more quickly.

* Online password managers allow users to store and manage multiple passwords securely in the cloud. They encrypt all stored data and allow users to access their passwords across different devices, making it easier to use strong and unique passwords.

* Virtual Private Networks (VPNs) protect user privacy by encrypting internet traffic and routing it through secure servers. This hides the user’s IP address and location, making it harder for websites or attackers to track online activity.

* Web application firewalls (WAFs) are used to protect websites from attacks such as SQL injection or cross-site scripting. They filter incoming traffic and block malicious requests before they reach the server.

* Email spam filters analyse incoming emails using cloud-based systems to detect spam, phishing, and malicious content. They automatically move suspicious emails to spam folders, reducing the risk of users interacting with harmful messages.

---

### Mitigations (high-level recommendations)

Using trusted tools, enabling extra security features, and staying informed about threats helps improve online security.

---

### Reflection

This task showed that online security tools are essential but must be combined with user awareness.
