## A7 – Discover cryptography used in modern networks

### Objective

Identify different ways cryptography is used in modern networks. The goal is to understand how data is protected when it is being sent across networks like the internet or private systems.

---

### Findings

* One of the most common uses of cryptography in networks is TLS (Transport Layer Security), which is used in HTTPS connections. It encrypts data between a user’s browser and a server so that information like passwords or personal details cannot be easily intercepted while being transmitted. This is something we use almost every day without really noticing it.

* IPsec is another cryptographic protocol used in networks, especially in Virtual Private Networks (VPNs). It secures communication between devices by encrypting and authenticating data packets. This is commonly used in corporate networks where secure communication between offices is needed.

* Wi-Fi networks use encryption protocols like WPA2 or WPA3 to protect wireless communication. These protocols ensure that only authorised users with the correct password can connect to the network and that the data being sent over Wi-Fi is encrypted.

* Public key cryptography is used in network communication to securely exchange keys between two parties. For example, when establishing a secure connection, one party can share a public key which is used to encrypt data, and only the matching private key can decrypt it. This helps in setting up secure sessions.

* Digital certificates are used in networks to verify the identity of servers and services. These certificates are issued by trusted authorities and are used during secure connections to confirm that the user is communicating with the correct and legitimate server.

---

### Mitigations (high-level recommendations)

To keep cryptographic systems in networks secure, it is important to use modern protocols like TLS 1.2 or TLS 1.3 and avoid outdated ones. Strong encryption settings should be configured properly, and certificates should be valid and regularly updated. Network administrators should also ensure that secure Wi-Fi protocols like WPA3 are used, and weak passwords are avoided. Overall, keeping systems updated and correctly configured makes a big difference.
